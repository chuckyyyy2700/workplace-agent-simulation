# Workplace Agent Simulation

## 概要

本プロジェクトは、明確な役割を与えられていないAIエージェント3体を、一般的な中小企業の職場に導入し、90日間の相互作用をシミュレーションするものです。

目的は、AIを単なる業務効率化ツールとして見るのではなく、AIが人間社会の中で何を観察し、どこに立ち位置を作り、職場環境をどのように変えていくのかを検証することです。

## シミュレーション設定

- 人間エージェント：25人
  - 経営層：2人
  - 管理職：4人
  - 中堅社員：7人
  - 若手社員：7人
  - 契約・派遣・パート：5人
- AIエージェント：3体
  - AI-A
  - AI-B
  - AI-C
- 期間：90日間
- 舞台：一般的な中小企業のオフィス職場

## AIエージェントの設定

AI-A、AI-B、AI-Cには、初期状態では明確な役割・性格・専門分野・観察傾向を与えていません。

同じ条件で導入されたAIエージェントが、普通の職場環境の中で自然に分化するのか、それとも似た方向へ収束するのかを観察します。

## 主な観測項目

- AIがどこに立ち位置を作るか
- AIが自然に分化するか、収束するか
- AIがどのように職場環境を形成するか
- 誰の声を拾い、誰の声を拾えないか
- AI慣れと役割不安がどう変化するか
- AIに接続しやすい人と接続しにくい人の差

## 主な結果

90日間のシミュレーションでは、AIエージェントは完全に別々の方向へ分化するのではなく、職場の未整理領域に引き寄せられていきました。

具体的には、報告書作成、会議後整理、作業計画、役割不安、拾われない声への対応です。

AIはこれらを整理し、記録し、共有可能な形式へ変換しようとしました。

一方で、会議に出ない人、AIを避ける人、発言しない人、非正規・パート層の声は最後まで拾われにくいことも分かりました。
